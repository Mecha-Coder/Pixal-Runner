### **How to run**

```bash
# Windows
python -m venv env
.\env\Scripts\activate
python -m pip install -r requirement.txt
python Pixel_Runner.py

# Linux distro
python3 -m venv env
source env/bin/activate
python3 -m pip install -r requirement.txt
python3 Pixel_Runner.py

```
